Guía de puntuación

Significado de la puntuación

El puntuación del Rice Purity Test va de 0 a 100. Un numero mas alto significa menos experiencias marcadas; uno mas bajo significa mas experiencias. La palabra “pureza” es historica y no debe tratarse como una clasificacion moral.

100

Pureza máxima posible

Este rango suele reflejar muy pocas experiencias marcadas. Tratalo como entretenimiento, no como una etiqueta.

95-99

Puntuación extremadamente alta

Este rango suele reflejar experiencias muy limitadas. No mide tu valor personal.

90-94

Puntuación muy alta

Este rango suele reflejar pocas experiencias. Cada persona avanza a su propio ritmo.

70-89

Puntuación alta

Este rango suele reflejar algunas experiencias y muchas casillas sin marcar.

50-69

Puntuación moderada

Este rango suele reflejar una mezcla mas amplia de experiencias.

20-49

Con experiencia

Este rango suele reflejar muchas experiencias. Miralo desde la seguridad, el respeto y el autocuidado.

10-19

Con mucha experiencia

Este rango suele reflejar muchisimas experiencias. No es un diagnostico ni un juicio.

1-9

Con experiencia extrema

Este rango suele reflejar casi todas las experiencias de la lista.

0

Experiencia completa

Este puntuación refleja todas las experiencias listadas. Es muy inusual y no define tu personalidad.

Preguntas frecuentes

¿Que puntuación es bueno?

No hay un puntuación universalmente bueno o malo. Depende de la edad, el contexto, la cultura y tus limites personales.

¿El puntuación me define?

No. Es solo una herramienta ligera de reflexion y un resultado para compartir si quieres.

¿Deberia compartirlo?

Compartelo solo si te sientes comodo. Nunca necesitas compartir tus respuestas personales.
