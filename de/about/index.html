About RicePurityTest.im

This project is a privacy-first, browser-based Rice Purity Test experience. The P2 version focuses on exact score pages, multilingual access, share cards, and a lightweight quiz interface.

The goal is to be faster, clearer, and more respectful than thin quiz clones.
